# Dungeon Crawler Learning Suite I (v1.0)

Welcome to the official repository for the Dungeon Crawler Learning Suite! This project is an open-source learning journey into Python OOP, game architecture, and secure data handling.

## Navigation Information
Please note that this front-page README serves only as a welcome and directory. The primary project roadmap and all lesson instructions are located within the suite folder.

### Repository Map:

* **Main Suite**: `dungeon_crawler_suite1/` — This folder contains the engine code, game files, and interactive guides.
* **Full Documentation**: Refer to `dungeon_crawler_suite1/readme.md` for the complete project guide and road map.
* **License**: The `license.txt` (CC BY 4.0) is located inside the `dungeon_crawler_suite1/` folder.
* **Legacy Archive**: `archive_zip/` — This folder contains the original 2013 C++ prototype for historical reference.

**Please navigate into the `dungeon_crawler_suite1` folder to begin.**
